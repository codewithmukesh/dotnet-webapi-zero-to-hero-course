# Soft Deletes in EF Core 10

Source code for the article: [Soft Deletes in EF Core 10 - Interceptors, Named Filters & Cascade Delete](https://codewithmukesh.com/blog/soft-deletes-efcore/)

## Running the Project

1. Start PostgreSQL:
```bash
docker-compose up -d
```

2. Run the API:
```bash
dotnet run --project SoftDeletes.Api
```

3. Open Scalar API Reference at `https://localhost:<port>/scalar/v2`
